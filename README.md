# Homebrew Tap

Custom Homebrew tap for tamtom tools.

## Installation

```bash
brew tap tamtom/tap
```

## Available Formulae

### gplay

CLI for Google Play Console - manage Android apps from the terminal.

```bash
brew install tamtom/tap/gplay
```

## Usage

After installation:

```bash
gplay --help
gplay auth login
gplay tracks list --package com.example.app
```

## Updating

```bash
brew update
brew upgrade gplay
```
